¿Que es CSS?

El CSS (Cascading Style Sheets), en español Hojas de Estilo en Cascada, fue desarrollado por W3C en 1996. Se le denomina Hojas de Estilos en Cascada porque las características se aplican de arriba a abajo mediante reglas que poseen un esquema prioritario.

¿Para qué sirve el CSS?

La tecnología CSS está diseñada para marcar la separación del contenido de las páginas web y la forma de presentación de estas. Lo que genera múltiples beneficios, como:

  1. Definir el estilo visual de todo un sitio web. Así, si cambiamos una página, cambiarán todas automáticamente

  2. Trabajar con estándares y separar (hasta cierto punto) la estructura de la presentación logrando un trabajo más definido

    3. Provee más flexibilidad y control en las especificaciones del sitio web

    Simplifica la creación de la página

    Evitar hacer archivos demasiado pesados

¿Por qué vale la pena implementar CSS en un sitio web?

Las ventajas de utilizar las Hojas de Estilo en Cascada son numerosas, algunas ya te las mencionamos anteriormente y otras te las diremos a continuación:

  • La facilidad de organizar y hacer cambios

  • El permitir que las páginas descarguen con mayor rapidez

  • la simplicidad para crear versiones dependiendo de los dispositivos (tablets, teléfonos inteligentes o dispositivos móviles, entre otros), etc.

Sintaxis del CSS

El CSS es un lenguaje basado en reglas: cada usuario define las reglas que especifican los grupos de estilos que van a aplicarse a elementos particulares o grupos de elementos de la página web. Por ejemplo: «Quiero que el encabezado principal de mi página se muestre en letras grandes de color rojo».

Antes de los dos puntos, tenemos la propiedad; y después, el valor. Las propiedades (en-US) CSS admiten diferentes valores, dependiendo de qué propiedad se esté especificando. En el ejemplo anterior, tenemos la propiedad color, que puede tomar varios valores de color. También tenemos la propiedad de font-size, que puede tomar varias unidades de tamaño como valor.
